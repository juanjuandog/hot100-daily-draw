# 秋招倒计时小组件

这个小组件会在 macOS 桌面右上角显示：

`秋招倒计时`
`还剩 X 天`

默认目标日是每年的 `9 月 1 日`。

## 启动

在项目根目录执行：

```bash
chmod +x /Users/juanjuandog/Documents/hot100/widget/launch-countdown-widget.command
/Users/juanjuandog/Documents/hot100/widget/launch-countdown-widget.command
```

## 特点

- 常驻右上角
- 始终置顶
- 支持拖动位置
- 自动按当天日期计算距离 `9/1` 还剩多少天

## 关闭

在“活动监视器”里结束 `hot100-countdown-widget` 进程即可。
